# Windows Software Playground

## 概要
Windows向けソフトを試作・検証したものをまとめたリポジトリ。  
小さな実験やアイデアのプロトタイプを記録している。

---

## 目的
- 技術検証
- アイデアの試作
- 実装パターンのストック

---

## 内容
- シンプルなツール
- UIの試作
- 動作検証用コード
- 途中で止まったプロジェクト（含む）

---

## 注意
- 基本的に試作レベル（完成度は低め）
- コードは実験的なものが多い

---

## 使い方
気になるプロジェクトを自由に参照してください。  
再利用・改変もOK（必要に応じて）。

---

## 🧠 補足
このリポジトリは「完成品」ではなく、  
試行錯誤や学習過程の記録を目的としています。
